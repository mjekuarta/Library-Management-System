# Library-Management-System

**Version 1.0.0**

LabCourse Project 2020

A desktop application that allows you to insert, edit, remove books, authors, and users while logging in as a user or an admin with privileged rights.


## Technologies used:
- Java and Java Swing
- Microsoft SQL
